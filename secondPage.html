"No sé que tan buena práctica sea poner una imagen dentro de una etiqueta de título"

Para realizar enlaces a otra páginas se usa la etiqueta a que es de contenido, con abre y cierre y que con la propiedad href se direcciona a otra página local o externa.

Existe la etiqueta nav para indicar cual es el panel de navegación de la página, en este caso encierra toda la lista

Para indicar que la lista sea horizontal en los estilos se indica con la etiqueta nav li la propiedad display: inline; para que cada item de la lista solo ocupe el espacio necesario de su palabra y que no de salto de línea

Para alterar la lista que sea en mayúsculas de color negro y en negrita se hace a través de CSS no de la etiquetas de HTML, por ejemplo para aumentar el espacio interno entre las palabras se usa la etiqueta margin: top right bottom left eligiendo a cual lado se quiere distanciar

Se tiene el archivo reset.css que es una colaboración de la comunidad para hacer que los estilos que el navegador crea automáticamente no choquen con los estilos hechos por el diseñador, este archivo los borra.

Para cambiar la posición de un elemento se cuenta con la etiqueta position y se puede ubicar por pixeles o porcentaje referente a los lados de la pantalla

Usando la etiqueta div puedo encapsular varios elementos para afectar los estilos de forma simultánea

Al usar auto en las propiedades de posición por ejemplo el navegador lo asume dependiendo de la pantalla y varía y parece siempre centrado en este caso
